# Auto Formatting Features for WYSIWYG Editors

This demonstrates how to use Froala's custom plugins to automate content formatting in the editor.

---

## Features

- **Consistent Typography and Style Application**: Change font sizes, font styles, and font weights according to predefined rules.
- **Auto Alignment and Spacing**: Format paragraph alignment and spacing.
- **Color Scheme and Branding Enforcement**: Set a brand color and apply it automatically to different elements in one click.

---

## Prerequisites

To run this example, you will need:

1. Froala Editor

---

## How to Run

### 1. Get the files from this repository and paste them into your project folder.

### 2. Run the application on your browser.

---

### How It Works

1. Paste formatted content into the editor. To better see the results, mix up the types of content (different headings, font styles, use quotes, ordered lists, etc.)
2. Click the "Auto Format" button on the toolbar.
3. Wait for the editor to process the content.

---

### Notes

- You can also try using Froala events to trigger auto formatting (on content changed, etc.).
- Try integrating AI into the formatting automation process.
